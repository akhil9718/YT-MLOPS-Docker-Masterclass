## ⚙️ Setup Instructions

### 🔹 1. Clone Repository

```bash
git clone https://github.com/akhil9718/YT-MLOPS-Docker-Masterclass.git
cd YT-MLOPS-Docker-Masterclass
```

---

### 🔹 2. Create Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

---

### 🔹 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 🔹 4. Run Application

```bash
python app.py
```

👉 Open in browser:

```
http://localhost:5000
```

---

## 🐳 Docker Setup

### 🔹 Build Docker Image

```bash
docker build -t mlops-docker-demo .
```

---

### 🔹 Run Docker Container

```bash
docker run -d -p 5000:5000 mlops-docker-demo
```
